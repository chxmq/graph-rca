# Roblox Incident

**Category:** Infrastructure
**Severity:** Critical
**Source:** https://blog.roblox.com/2022/01/roblox-return-to-service-10-28-10-31-2021/

## Description

73-hour outage caused by HashiCorp Consul cluster failure when enabling a new streaming feature under high load.

## Root Cause

Hashicorp consul cluster failure when enabling a new streaming feature under high load
