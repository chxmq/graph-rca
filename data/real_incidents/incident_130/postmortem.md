# Homebrew Incident

**Category:** Security
**Severity:** Medium
**Source:** https://web.archive.org/web/20210813020247/https://brew.sh/2018/08/05/security-incident-disclosure/

## Description

A GitHub personal access token with recently elevated scopes was leaked from Homebrew’s Jenkins that allowed access to `git push` on several Homebrew repositories.

## Root Cause

A GitHub personal access token with recently elevated scopes was leaked from Homebrew’s Jenkins that allowed access to `git push` on several Homebrew repositories
